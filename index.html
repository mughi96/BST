Binary Search Tree

In Computer Science, trees refer to a widely used abstract data structure that is hierachical in nature, containing a root node that spans into sub-trees with child nodes that are all linked.


Binary Search Trees are built upon the architecture of a basic binary search algorithm. Hence, Binary Search Trees enable quicker lookups, insertions and removal of nodes.

A Binary Search Tree has the following properties:


  • The left sub-tree of a node consists of nodes with integers that are less than the integer stored in the node itself.

  • The right sub-tree of a node consists of nodes with integers that are greater than the integer stored in the node itself.

  • The left and right sub-trees of a node should be Binary Search Trees themselves.

In order for a tree to be considered a Binary Search Tree, it must abide by the above properties. Hence, Binary Search Trees can only contain distinct values and no duplicate values.



How To Use This Tool

This tool is meant to demonstrate the basic working principles of a Binary Search Tree. As nodes are inserted, the Binary Search Tree will appear in the empty canvas on the left.


There are 2 main operations featured in this tool:
  1. Insert

  2. Search

Scroll down along this box for instructions on how to execute the operations.

Insert a Node

To insert a node into the Binary Search Tree:


  1. In the 'INSERT' input box, enter an integer of maximum 4 digits

  2. Press 'INSERT'

  3. The interger inserted will appear as a green node in the Binary Search Tree, with the path to the node highlighted in blue.

  4. The Insert operation is complete

Search for a Node

To search for a node in the Binary Search Tree:


  1. In the 'SEARCH' input box, enter an integer to be searched in the Binary Search Tree

  2. Press 'SEARCH'

  3. The Binary Search Tree executes a search and highlights the path to the node in red

  4. If the node is not found, a pop-up will appear to indicate that the node was not found in the tree.

  5. The Search operation is complete

Reset the Binary Search Tree

To erase all the nodes in the Binary Search Tree, press the "RESET BINARY SEARCH TREE" button at the top left.
